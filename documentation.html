Documentation FAQ

Common questions about using and contributing to AnimateItNow.

How do I add a new animation template?

Fork the repository, add your template in the templates/ folder, and submit a pull request.

Can I customize the theme?

Yes! Use the theme toggle in the navigation bar or modify CSS variables in :root.

Where can I find API documentation?

API documentation is available in the README.md and inline comments in script.js.

Who do I contact for support?

Email contact@animateitnow.com or join our Discord community.
