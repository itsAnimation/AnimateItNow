2D Transform Effects

Explore various 2D transform effects for buttons and elements with CSS animations.

Translate

Moves an element from its original position.
<!-- HTML -->
<div class="transform-demo">
  <button class="transform-demo-btn transform-translate">Translate</button>
</div>

<!-- CSS -->
.transform-translate:hover {
  transform: translate(40px, 0); /* Moves 40px right, 0px down */
}

Rotate

Rotates an element around its center.
<!-- HTML -->
<div class="transform-demo">
  <button class="transform-demo-btn transform-rotate">Rotate</button>
</div>

<!-- CSS -->
.transform-rotate:hover {
  transform: rotate(20deg); /* Rotates 20 degrees clockwise */
}

Scale X

Changes the width of an element.
<!-- HTML -->
<div class="transform-demo">
  <button class="transform-demo-btn transform-scale-x">Scale X</button>
</div>

<!-- CSS -->
.transform-scale-x:hover {
  transform: scaleX(1.5); /* Makes it 1.5 times wider */
}

Scale Y

Changes the height of an element.
<!-- HTML -->
<div class="transform-demo">
  <button class="transform-demo-btn transform-scale-y">Scale Y</button>
</div>

<!-- CSS -->
.transform-scale-y:hover {
  transform: scaleY(1.5); /* Makes it 1.5 times taller */
}
