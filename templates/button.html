üéÜ Rainbow Glow

CSS + HTML 
.rainbow-btn {
  color: #fff;
  padding: 1em 2em;
  border: none;
  border-radius: 2em;
  font-weight: 600;
  cursor: pointer;
  position: relative;
  z-index: 1;
  background: #333;
  transition: all 0.3s ease;
}

.rainbow-btn::after {
  content: "";
  z-index: -1;
  position: absolute;
  width: 100%;
  height: 100%;
  background-color: #333;
  left: 0;
  top: 0;
  border-radius: 2em;
}

.rainbow-btn::before {
  content: "";
  background: linear-gradient(45deg, #ff1616, #e4d20a, #6adf0a, #10cd8b, #09c2df, #371ddd, #dc13a9, #de1212);
  position: absolute;
  top: -2px;
  left: -2px;
  background-size: 600%;
  z-index: -1;
  width: calc(100% + 4px);
  height: calc(100% + 4px);
  filter: blur(1em);
  animation: rainbow-glow 20s linear infinite;
  border-radius: 2em;
  opacity: 0;
  transition: opacity 0.3s ease;
}

@keyframes rainbow-glow {
  0% { background-position: 0 0; }
  50% { background-position: 400% 0; }
  100% { background-position: 0 0; }
}

.rainbow-btn:hover {
  transform: scale(1.08);
}

.rainbow-btn:hover::before {
  opacity: 1;
}

<button class="rainbow-btn">Hover Me</button>
